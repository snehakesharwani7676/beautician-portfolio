# Sneha Kesharwani - Professional Beautician Portfolio

A modern, elegant, and responsive portfolio website for a professional beautician, makeup artist, and cosmetologist.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, elegant design with rose-gold, pink, and gold color scheme
- **Smooth Animations**: Scroll animations and hover effects throughout
- **Portfolio Gallery**: Organized sections for makeup, hair, and skin/nails work
- **Booking System**: Integrated WhatsApp booking with form validation
- **Contact Form**: Easy-to-use contact form with email integration
- **Google Maps**: Embedded location map

## Sections

1. **Home** - Hero section with professional introduction
2. **About** - Detailed background and expertise
3. **Skills** - Categorized skill cards (Makeup, Hair, Nails, Skin)
4. **Education** - Timeline of qualifications and training
5. **Portfolio** - Filterable gallery of work samples
6. **Booking** - Appointment booking form with WhatsApp integration
7. **Contact** - Contact information and form

## How to Use

1. Open `index.html` in a web browser
2. Replace placeholder images with actual photos:
   - Hero section: Add professional photo
   - About section: Add profile photo
   - Portfolio section: Add work samples

## Customization

### Adding Photos

Replace the `.image-placeholder` divs with actual images:

```html
<!-- Replace this: -->
<div class="image-placeholder">
    <i class="fas fa-user-circle"></i>
    <p>Add Your Photo</p>
</div>

<!-- With this: -->
<img src="your-photo.jpg" alt="Description">
```

Modify colors in `styles.css` under `:root` variables:
```css
:root {
    --rose-gold: #b76e79;
    --pink: #ff69b4;
    --light-pink: #ffb6c1;
    --gold: #d4af37;
}
```
