Images folder structure for beautician-portfolio

This directory contains categorized image folders used by the site.

images/
├── makeup/
│   ├── self-makeup/
│   ├── teen-makeup/
│   ├── reception-makeup/
│   ├── engagement-makeup/
│   ├── bridal-makeup/
│   ├── glam-makeup/
│   ├── fantasy/
│   ├── bronze-tan/
│   └── model-bride/
├── hair/
│   ├── hair-do/
│   ├── keratin/
│   ├── rebonding/
│   ├── botox/
│   ├── smoothening/
│   └── hairstyles/
└── skin-nails/
    ├── nail-art/
    ├── nail-extensions/
    ├── facials/
    ├── manicure-pedicure/
    └── waxing/

Each leaf folder contains a .gitkeep file so empty folders are tracked until you add images.
